Face Detection and Clustering
Jio Platforms Limited Internship Project
Overview

This project focuses on building an end-to-end Face Detection and Clustering Pipeline capable of automatically detecting faces from large image collections, extracting facial features, and grouping images belonging to the same individual without prior identity labels.

The system is designed to process thousands of images efficiently and generate meaningful face clusters that can be used for photo organization, media analytics, identity grouping, and downstream facial recognition workflows.
